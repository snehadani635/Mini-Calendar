# Mini Calendar

A simple and responsive Mini Calendar built using HTML, CSS, and JavaScript. The application automatically displays the current date, day, month, and year using JavaScript's Date object.

## Features

- Displays current date automatically
- Shows current day, month, and year
- Modern and attractive user interface
- Responsive design
- Real-time date updates on page load
- Clean and beginner-friendly code

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

1. The application creates a new Date object.
2. JavaScript fetches:
   - Current Date
   - Current Day
   - Current Month
   - Current Year
3. The values are displayed dynamically on the webpage.

## Concepts Used

- DOM Manipulation
- JavaScript Date Object
- Arrays
- getDate()
- getDay()
- getMonth()
- getFullYear()
- Dynamic Content Update
